Free Subnet Calculator & Network Planning Tools

      
Network Sloth is a free, browser-based toolkit built for network engineers who need fast, accurate subnetting without fuss. Every calculation runs locally in your browser — no data is sent to any server, no account is required, and there is no rate limiting. Enter a network address, choose your prefix length and calculation mode, and get subnet addresses, broadcast addresses, usable host ranges, Cisco wildcard masks and CSV export in one click.

    


    


      

        
CIDR Subnet Calculator

        
Calculate subnets from any IPv4 address using CIDR notation — from /8 to /30. Choose between three modes: split a network into a set number of subnets, find the right subnet size for a given number of hosts, or specify a custom prefix length directly. Results include the network address, subnet mask, CIDR prefix, Cisco wildcard mask, first usable host, last usable host, broadcast address and usable host count for every subnet.

      


      

        
VLSM Network Scenario Planner

        
The Scenario Planner uses Variable Length Subnet Masking (VLSM) to allocate the most efficient subnet size to each VLAN or network segment automatically. Choose from pre-built templates — small corporate network, medium enterprise, home lab, branch office or data centre — or define your own VLANs with custom host counts. The engine sorts segments largest-first and packs them into your chosen address space with no wasted addresses.

      


      

        
DHCP Scope Generator

        
Generate ready-to-paste DHCP configuration for every subnet in your plan. Supported platforms include Cisco IOS (ip dhcp pool), Windows Server DHCP (PowerShell Add-DhcpServerv4Scope), ISC DHCP (dhcpd.conf), Kea DHCP4 (JSON) and MikroTik RouterOS. Set DNS servers, domain suffix, lease duration and exclusion ranges per scope. Import subnets directly from the Scenario Planner or the Manual Calculator.

      


      

        
Firewall Rule Generator

        
Generate inter-VLAN firewall rules from your network plan without writing ACLs by hand. Choose a policy preset — Strict (deny all inter-VLAN, permit internet), Relaxed (permit all), or Custom (define rules by source VLAN, destination, protocol and port) — then select your output platform: Cisco IOS named extended ACLs, pfSense or OPNsense XML, or Juniper EX/QFX firewall filters. Output includes interface application commands and is ready to copy directly into your device.

      


    


    
    

      
Frequently Asked Questions


      

        What is CIDR notation?
        
CIDR (Classless Inter-Domain Routing) notation expresses an IP network as a base address and prefix length — for example 192.168.1.0/24. The number after the slash tells you how many bits form the network portion. A /24 gives 256 addresses (254 usable hosts); a /25 gives 128; a /30 gives 4 (2 usable — useful for point-to-point WAN links).

      


      

        What is a wildcard mask and how is it different from a subnet mask?
        
A subnet mask uses binary 1s to mark the network bits (255.255.255.0 for a /24). A wildcard mask is the bitwise inverse — 0s mark the network bits and 1s mark the host bits (0.0.0.255 for a /24). Cisco IOS uses wildcard masks in access control lists (ACLs) and OSPF network statements. Network Sloth displays the Cisco wildcard mask alongside every subnet result.

      


      

        What is VLSM and why should I use it?
        
Variable Length Subnet Masking (VLSM) lets you carve different-sized subnets from a single address block — a /30 for a WAN link, a /25 for a staff VLAN, a /28 for servers — rather than wasting a fixed block on every segment. This maximises address efficiency, which matters whether you are working with public IP space or keeping RFC 1918 private ranges tidy across many sites.

      


      

        How do I calculate subnets and usable hosts?
        
Subnets created by borrowing n bits from the host portion = 2n. Usable hosts per subnet = 2h − 2, where h is the number of remaining host bits (the −2 accounts for the network address and broadcast address). Example: splitting a /24 into /26s borrows 2 bits → 4 subnets, each with 26 − 2 = 62 usable hosts. Network Sloth calculates all of this automatically.

      


      

        Is Network Sloth completely free?
        
Yes. Every tool on Network Sloth — the subnet calculator, VLSM planner, DHCP scope generator and firewall rule generator — is free to use with no account, no rate limits and no expiry. All calculations run in your browser; nothing is sent to a server.
